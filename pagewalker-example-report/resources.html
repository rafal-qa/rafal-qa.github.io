List of unique resources downloaded by all visited pages
  • Internal - From same domain
  • External - From other domain, subdomain
  • Number of requests - How many times this resource was used on all visited pages
  • Failed requests - Number of requests with error (HTTP status 4xx/5xx or other problem)
    Click error count to open list of example pages on which the error occurred
  • Cached requests - How often this resource was requested from cache
  • Resource size - Average resource size calculated from non-cached requests
    Compressed size if compression on server is enabled
  • Non-cached avg. load time - Average resource load time calculated from non-cached requests
  • Unfinished requests - When the browser send request and go to the next page without waiting for a response
    For example asynchronous, non-blocking requests
    We are unable to determine if this is an error
  • Blacklist status (only External) - Tells if the domain is blacklisted because of malware, scam
